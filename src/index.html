Landing-page «Suppke»

Одностраничный сайт. Реализована корзина, возможность добавления/удаления и подсчета стоимости товаров. Добавлены слайдер, сортировка каталога товаров по категориям, разичные анимации на css, javascript. Отправка заказа, а также сообщения через форму обратной связи, происходит в формате json на фейк-сервер jsonplaceholder.

Стэк: javaScript, HTML, SCSS. Без использования js-библиотек.
